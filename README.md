# ChatGPT clone

> a demo of the App

  <img src="screenshots/demo.gif" width="800px" alt="android icon"/>


## install

```bash
npm install
```

## Configuration
1. `cd server`
2. rename `.env.example` to `.env`
3. add your openai api key inside `.env`

## run
```bash
npm run dev
```


***Tech used***
  - openai
  - react with vite
  - tailwindcss
  - react-icons
  - react-markdown
  - express
